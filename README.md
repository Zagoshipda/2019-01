## Install & Run
- 😘Front-End
```sh
cd client
npm install
npm start
```
- 🏗Back-End
```sh
cd server
npm install
npm start
```
- 🐳By Docker-compose
```sh
docker-compose up
```

## ⛑Run tests

```sh
npm test
```
